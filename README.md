# perros-app-js

Aplicación web hecha con JavaScript vainilla que consulta la Dog API para mostrar fotos de perros según la raza ingresada.

## Funcionalidad

- El usuario escribe una raza de perro.
- La aplicación consulta la API usando fetch.
- Si la búsqueda sale bien, se muestran 6 fotos en pantalla.
- Si la raza no existe o hay un error de red, se informa el problema en el DOM y también en la consola.

## Tecnologías

- HTML
- CSS
- JavaScript
- Dog API: https://dog.ceo/dog-api/
